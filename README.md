# Keetbot
**Keetbot** is a very simple python library for making your [keet messenger](https://keet.io/) bots 
### Installing
With pip
```pip install keet```

From source
```pip install git+https://github.com/lmemq/keetbot.git```

Upgrade
```pip install keet --upgrade```
